## 1. Контекст
Использование стандартной библиотеки потоков C++ (`std::thread`) позволяет реализовать параллелизм без зависимости от внешних библиотек (OpenMP, TBB). Эта версия демонстрирует навыки ручного управления жизненным циклом потоков и явного разделения данных между ними.

## 2. Постановка задачи
Реализовать параллельное умножение матриц в формате CRS, используя `std::thread`. Проверить масштабируемость алгоритма при увеличении числа потоков и сравнить результаты с последовательным эталоном [SEQ](../seq/report.md).

## 3. Базовый алгоритм
Для обеспечения потокобезопасности без использования блокировок (mutex) применен двухпроходный алгоритм:
1. **Первый проход (Символический):** Потоки параллельно вычисляют количество ненулевых элементов в отведенных им строках. Результаты заносятся в локальные инкременты `row_ptr`.
2. **Префиксная сумма:** В основном потоке выполняется последовательный расчет указателей строк для формирования структуры матрицы.
3. **Второй проход (Вычислительный):** Потоки вычисляют сами значения элементов и индексы столбцов, записывая их в заранее выделенные участки общей памяти.

## 4. Схема распараллеливания
- **Разбиение:** Матрица A делится на блоки строк (chunks). Каждый блок обрабатывается отдельным потоком. Границы вычисляются как `start = t * chunk` и `end = (t + 1) * chunk`.
- **Управление потоками:** Создается вектор `std::vector<std::thread>`. Потоки запускаются через лямбда-функции.
- **Синхронизация:** Основной поток ожидает завершения всех рабочих потоков с помощью вызова `.join()` после каждого прохода.
- **Локальные данные:** Каждый поток владеет собственными буферами `marker`, `acc` и `used`, что исключает гонки данных.

void MaslovaUMultMatrSTL::ComputeValuesSTL(int rows_a, int cols_b, int num_threads) {
  std::vector<std::thread> threads;
  int chunk = rows_a / num_threads;
  for (int thread_idx = 0; thread_idx < num_threads; ++thread_idx) {
    int start = thread_idx * chunk;
    int end = (thread_idx == num_threads - 1) ? rows_a : (thread_idx + 1) * chunk;
    if (start < end) {
      threads.emplace_back([&, start, end, cols_b]() {
        std::vector<double> acc(cols_b, 0.0);
